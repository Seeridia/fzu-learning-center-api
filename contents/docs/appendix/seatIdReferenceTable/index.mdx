对于同一个座位，我们有下面的这三个编号

- `spaceName`: 在现场看到的编号的标识
- `id/spaceId`: 用于接口参数的编号
- `spaceCode`: 不知道

例如下面是同一个座位

- `spaceName`: "222"
- `id`: 1592
- `spaceCode`: "4B214"（4 楼 B 区）

另外，目前已经从官方的 3D 图中获取到了下面两个全新的编号，下面这个编号和上面的三个编号是同一个座位

- `mesh`: 209
- `mapName`: 214 (和 `spaceCode` 的位置后的数字相同)

以及我也拿到了座位的坐标（3D 到 2D 去掉 Z 轴），可以用于绘图

目前下面这两个 json 便是所有座位的各个坐标和编号的对照表（累坏我了）
