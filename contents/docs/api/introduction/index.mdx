一些必要的说明：
1. `spaceId` 或 `id` 是不同于我们看到的编号，是另一个号码，我们熟知的编号一般是 `spaceName`，对照表放在 [这里](/docs/appendix/seatIdReferenceTable)，或可以从 [这里](https://raw.githubusercontent.com/Seeridia/fzu-learning-center-api/main/source/seatIdReferenceTable.csv) 下载，你也可以在 [这里](/docs/appendix/seatIdConversionTool) 找到一个转换工具

2. 接口的 `token` 是通过登录获取的，具体获取方式请查看 [这里](/docs/appendix/tokenObtain)

3. 接口存在速率限制，过快的请求会被拒绝，具体限制未知
